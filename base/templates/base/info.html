What is StudioLog?
StudioLog allows musicians to keep track of the hours they spend on their instruments. It lets users record the details of each practice session, as well as the instruments and pieces involved. Users can then easily access and filter through this information, and even view some summary statistics!
Use Cases
This website can not only be used by aspiring musicians to monitor their progress, but also by music teachers and students to record the details of each lesson and store notes for each lesson, instrunment, or piece.
Note: This website is a passion project meant for my own learning, and is not intended for commercial distribution or use. View the code here.
